# Bhargesh Patel — Portfolio

Modern minimal portfolio built with [Astro](https://astro.build), Tailwind CSS, and [Impeccable](https://impeccable.style) design guidance.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
npm run preview
```

## Design context

- `PRODUCT.md` — brand brief, users, anti-references
- `DESIGN.md` — visual tokens and component rules

## Deploy

Configured for Netlify (`netlify.toml`). Publish directory: `dist`.

## Quality

```bash
npx impeccable detect src/
```
